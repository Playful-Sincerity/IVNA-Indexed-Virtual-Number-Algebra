  This is a known result in NSA (Goldblatt 1998, Theorem 5.7.2).
  Proof sketch:
    ln((1 + x*ε₀)^{y/ε₀}) = (y/ε₀) * ln(1 + x*ε₀)
    Using Taylor: ln(1 + h) = h - h²/2 + h³/3 - ...
    = (y/ε₀) * (x*ε₀ - x²*ε₀²/2 + x³*ε₀³/3 - ...)
    = xy - x²y*ε₀/2 + x³y*ε₀²/3 - ...
    st(this) = xy
    Therefore st((1 + x*ε₀)^{y/ε₀}) = e^{xy}

  Symbolic verification of leading term:
    (y/ε₀) * ln(1 + x*ε₀) ≈ (y/ε₀) * (x*ε₀ - x²ε₀²/2 + x³ε₀³/3)
